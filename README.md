# KaraGroup

Экосистема для управления заказами по прокачке игровых аккаунтов.

## Структура

```
karagroup/
├── backend/        — API сервер (FastAPI + SQLite)
├── admin-app/      — Десктоп-приложение для администратора (Electron + React)
└── worker-app/     — Десктоп-приложение для качеров (Electron + React)
```

---

## Backend

### Требования
- Python 3.10+

### Запуск

```bash
./start.sh
```

Или вручную:

```bash
cd backend
python3 -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

Сервер запустится на `http://localhost:8000`
Документация API: `http://localhost:8000/docs`

### Первый запуск

При первом старте автоматически создаётся администратор:

```
Логин:  admin
Пароль: admin123
```

> Смени пароль через раздел «Качеры» → создай нового админа и удали дефолтного.

### Модели данных

| Модель | Описание |
|--------|----------|
| `User` | Пользователи (admin / worker), процент выплаты |
| `Product` | Услуги с ценой (прокачка, фарм и т.д.) |
| `Order` | Заказы — источник, статус, исполнитель, цена |
| `Transaction` | Выплаты качерам по каждому заказу |
| `ChatMessage` | Сообщения чата внутри заказа |

### Статусы заказа

```
paid → in_progress → completed → confirmed
```

- `paid` — заказ создан, ждёт качера
- `in_progress` — качер взял в работу
- `completed` — качер отметил выполненным
- `confirmed` — администратор подтвердил

### Бизнес-правила

- Качер не может взять более **3 заказов** одновременно
- Выплата качеру рассчитывается в момент взятия заказа: `цена × процент_качера / 100`
- Источники заказов: `FunPay`, `Telegram`, `Другое`

## Admin App

Десктоп-приложение для администратора.

### Разделы

- **Дашборд** — выручка по дням, статусы заказов, источники, топ услуг, статистика по каждому качеру
- **Заказы** — создание с ручным ID, источником и ценой; смена статуса; фильтры; удаление
- **Услуги** — добавление/редактирование/скрытие услуг с ценой
- **Качеры** — создание аккаунтов, настройка процента выплаты, включение/отключение
- **Финансы** — список транзакций, кнопка «Выплатить» для каждой

Чат с качером открывается прямо на странице заказов — кнопка **Чат** в строке заказа.

### Запуск (dev)

```bash
cd admin-app
npm install
npm run dev
```

### Сборка

```bash
npm run dist:win    # Windows (.exe installer)
npm run dist:mac    # macOS (.dmg)
npm run dist:all    # оба сразу
```

Готовые файлы окажутся в `admin-app/dist-electron/`.

## Worker App

Десктоп-приложение для качеров.

### Разделы

- **Доступные заказы** — пул свободных заказов, автообновление каждые 15 сек, кнопка «Взять заказ»
- **Мои заказы** — активные заказы и история, кнопка «Выполнен», чат с администратором
- **Мои заработки** — история транзакций с суммами и статусами выплат

В сайдбаре отображается текущая ставка качера (%).

### Запуск (dev)

```bash
cd worker-app
npm install
npm run dev
```

### Сборка

```bash
npm run dist:win
npm run dist:mac
npm run dist:all
```

## Быстрый старт (всё вместе)

1. Запусти бэкенд:
   ```bash
   ./start.sh
   ```

2. В отдельном терминале запусти нужное приложение:
   ```bash
   cd admin-app && npm install && npm run dev
   # или
   cd worker-app && npm install && npm run dev
   ```

3. Войди в админку (`admin` / `admin123`), создай услуги и качеров.

4. Качер входит в свой аккаунт через worker-app и берёт заказы из пула.
